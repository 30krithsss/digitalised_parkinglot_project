🔧 BATCH SCRIPTS (5 Files) - For Windows
─────────────────────────────────────────────────────────────────────────────

1. check_system.bat
   ├─ Verifies Node.js (v16+)
   ├─ Checks npm
   ├─ Verifies MongoDB availability
   ├─ Confirms dependencies installed
   ├─ Shows system readiness status
   └─ Recommended to run FIRST

2. run_backend.bat
   ├─ Starts Express server on port 5000
   ├─ Connects to MongoDB
   ├─ Initializes Socket.io
   ├─ One-click startup
   └─ Run this in Terminal 1

3. run_frontend.bat
   ├─ Starts React app on port 3000
   ├─ Opens browser automatically
   ├─ Enables hot reloading
   ├─ One-click startup
   └─ Run this in Terminal 2

4. seed_database.bat
   ├─ Creates 100 parking slots
   ├─ Initializes 10 zones (A-J)
   ├─ Sets all slots to 'available'
   ├─ One-click database initialization
   └─ Run this in Terminal 3 (after apps start)

5. start_all.bat
   ├─ Displays startup instructions
   ├─ Shows all access URLs
   ├─ Quick reference information
   └─ Use this for reference
