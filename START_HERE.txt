🚀 3-MINUTE QUICK START
════════════════════════════════════════════════════════════════════════════

Prerequisites:
  ✓ Node.js (v16+) from https://nodejs.org
  ✓ MongoDB running OR MongoDB Atlas cloud connection

Step 1: Open Terminal & Run MongoDB
  > mongod
  Keep this open

Step 2: Open NEW Terminal - Start Backend
  > Double-click: run_backend.bat
  OR: cd backend && npm run dev

Step 3: Open NEW Terminal - Start Frontend
  > Double-click: run_frontend.bat
  OR: cd frontend && npm start

Step 4: Open NEW Terminal - Seed Database (wait 30 seconds for apps to start)
  > Double-click: seed_database.bat
  OR: curl -X POST http://localhost:5000/api/slots/seed/all

Step 5: Open Browser
  > http://localhost:3000/dashboard

Done! You're now running the complete parking management system.

💾 DATABASE STRUCTURE
════════════════════════════════════════════════════════════════════════════

Total Parking Spaces: 100
  • 10 Zones (A through J)
  • 10 Slots per zone
  • Each zone has 10 numbered slots (01-10)

Example Slot IDs: A-01, A-02, ..., A-10, B-01, ..., J-10

Database Collections:

1. Slots (Parking information)
   - slotNumber, zone, status
   - licensePlate, entryTime, exitTime
   
2. Users (User accounts)
   - email, password (hashed)
   - role (user/admin), subscription
   
3. Payments (Billing records)
   - userId, slotNumber, amount
   - entryTime, exitTime, duration


💡 PRICING EXPLAINED
════════════════════════════════════════════════════════════════════════════

Base Rate: ₹20 per hour

PEAK HOURS (8AM-10AM, 5PM-8PM):
  ₹20 × 1.5 = ₹30/hour
  
OFF-PEAK HOURS:
  ₹20 × 1.2 = ₹24/hour

Example Bill:
  Entry: 8:00 AM (peak)
  Exit: 10:30 AM (mixed)
  Duration: 2.5 hours
  Calculation: 2 hrs @ ₹30 + 0.5 hrs @ ₹24 = ₹72

🚨 TROUBLESHOOTING QUICK FIXES
════════════════════════════════════════════════════════════════════════════

PORT 5000/3000 IN USE:
  Windows: netstat -ano | findstr :5000
           taskkill /PID <number> /F

MONGODB WON'T CONNECT:
  Solution 1: Run 'mongod' command
  Solution 2: Use MongoDB Atlas (cloud)

FRONTEND CAN'T REACH BACKEND:
  • Ensure backend is running on port 5000
  • Check browser console (F12) for errors
  • Restart both services

3D DASHBOARD NOT RENDERING:
  • Clear browser cache (Ctrl+Shift+Delete)
  • Try Chrome browser
  • Update GPU drivers
