🎯 WHAT YOU HAVE
════════════════════════════════════════════════════════════════════════════

A fully automated, AI-powered parking lot management system built with:

FRONTEND:  React 19 + Three.js (3D visualization)
BACKEND:   Node.js/Express + Socket.io (Real-time)
DATABASE:  MongoDB (100 parking slots)

Features:
  ✅ AI License Plate Recognition (OCR)
  ✅ 3D Real-time Dashboard
  ✅ Smart Lock Control (Simulated)
  ✅ Dynamic UPI Payment System
  ✅ Admin Analytics & Controls
  ✅ Voice Commands
  ✅ JWT Authentication
  ✅ Real-time Socket.io Updates

🚀 GETTING STARTED (5 STEPS)
════════════════════════════════════════════════════════════════════════════

Step 1: VERIFY SYSTEM
  > check_system.bat
  (Checks Node.js, npm, MongoDB, and dependencies)

Step 2: START MONGODB
  > mongod
  (Keep this terminal open. Or use MongoDB Atlas cloud)

Step 3: START BACKEND SERVER (Terminal 1)
  > run_backend.bat
  OR: cd backend && npm run dev
  ✓ Listening on http://localhost:5000

Step 4: START FRONTEND (Terminal 2)
  > run_frontend.bat
  OR: cd frontend && npm start
  ✓ Listening on http://localhost:3000

Step 5: SEED DATABASE (Terminal 3)
  > seed_database.bat
  OR: curl -X POST http://localhost:5000/api/slots/seed/all
  ✓ Creates 100 parking slots

💳 PRICING MODEL
════════════════════════════════════════════════════════════════════════════

Base Rate: ₹20 per hour

PEAK HOURS (8AM-10AM, 5PM-8PM):
  Rate = ₹20 × 1.5 = ₹30/hour
  Examples:
  • 1 hour:   ₹30
  • 2 hours:  ₹60
  • 3 hours:  ₹90

OFF-PEAK HOURS:
  Rate = ₹20 × 1.2 = ₹24/hour
  Examples:
  • 1 hour:   ₹24
  • 2 hours:  ₹48
  • 3 hours:  ₹72

Mixed Duration:
  Entry: 8:00 AM (peak), Exit: 10:30 AM (off-peak)
  2 hrs peak @ ₹30 + 0.5 hr off-peak @ ₹24 = ₹72

📊 DATABASE SCHEMA
════════════════════════════════════════════════════════════════════════════

SLOTS Collection (100 documents):
{
  slotNumber: "A-01",          // Unique identifier
  zone: "A",                   // Zone A-J
  status: "available",         // available|occupied|reserved
  licensePlate: null,          // Car plate if occupied
  entryTime: null,             // When car entered
  exitTime: null,              // When car left
  timestamps: {...}            // Created/updated
}

USERS Collection:
{
  email: "user@example.com",
  password: "$2a$10$...",       // bcrypt hashed
  role: "user",                // user|admin
  subscription: "free",        // free|premium
  timestamps: {...}
}

PAYMENTS Collection:
{
  userId: ObjectId,
  slotNumber: "A-01",
  entryTime: "2024-01-15T08:00Z",
  exitTime: "2024-01-15T10:30Z",
  duration: 2.5,               // hours
  baseRate: 20,                // ₹/hour
  peakMultiplier: 1.5,
  totalAmount: 75,             // ₹
  status: "paid",              // pending|paid
  timestamps: {...}
}

Q: How do I start the system?
A: Run check_system.bat, then run_backend.bat, run_frontend.bat, seed_database.bat

Q: Do I need MongoDB installed?

🆘 TROUBLESHOOTING
════════════════════════════════════════════════════════════════════════════

Problem                 Solution
─────────────────────────────────────────────────────────────
MongoDB fails           Start mongod OR use MongoDB Atlas
Port in use            Kill process using netstat & taskkill
Dependencies missing   npm install in backend/ & frontend/
Frontend won't load    Clear browser cache (Ctrl+Shift+Delete)
3D not rendering       Try Chrome, update GPU drivers
Backend can't start    Check .env, ensure node_modules exist
Socket.io errors       Restart backend, check network
